atsuageのインストール

atsuageのインストール方法について説明します。

atsuageのインストールと使用のために、いくつかのソフトウェアとライブラリを使用します。以下の順に、インストールをしていくことをオススメします。

  1. Roswellのインストール
  2. Rosaのインストール
  3. atsuageのインストール

1. Roswellのインストール

Roswellのgithub wikiを参考にインストールしてください。

2. Rosaのインストール

atsuageは、その内容を表すtextファイルの形式にRosaのそれを採用しています。

$ ros install t-sin/rosa

でRosaをインストールしてください。

3. atsuageのインストール

$ ros install myaosato/atsuage

で、atsuageがインストールされます。
