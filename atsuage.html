静的サイト生成 atsuage 紹介

静的サイト生成のためのソフトウェア atsuage の紹介ページです。

atsuageは、Common Lispで書かれた、静的サイト生成を行うソフトウェアです。タグ付きのテキストファイルと、HTMLの様なS式から、HTMLファイルを作成して、貴方がWEBサイトを作成することをお手伝いします。 このWEBサイトもatsuageを使用して作成しています。

atsuageのソースコードは、以下のページから入手することが出来ます。

myaosato/atsuage: simple static site generator - GitHub

atsuageのインストールと使い方については、以下のページを参考にしてください。

  • atsuageのインストール
  • atsuageの使い方

atsuageとは

atsuageは、Common Lispで書かれた、静的サイト生成を行うソフトウェアです。ページ間の関係、例えば各記事をまとめたページと各記事や順番付けされた各記事などを、簡単にですが管理することも出来ます。

Lispのplistの様な形でタグ付けがされたテキストファイル(以下、テキストファイル)と、S式のようなHTMLのようなテンプレート用ファイル(以下、テンプレートファイル)を合わせてHTMLファイルを作ります。 テキストファイルの内容で、テンプレートファイルの中身を埋めていくようなイメージです。 この時に簡単なループ処理の様なものや、主に処理しているテキストファイルの内容から辿って別のテキストファイルの内容を用いるなどのことも可能です。

詳しい使い方は、atsuageの使い方をご覧ください。
