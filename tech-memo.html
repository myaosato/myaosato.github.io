コンピュータ関連のメモ

技術的メモという名前にしようかと思ったが、それはそれで、何を指し示すのかわからないので、コンピュータ関連のメモとした。

まあ、流石にそういうの用意しないとアレだよなぁと思って。鳥頭だからね。三歩進んで二歩忘れる。

とりあえず、ここにダラダラ書いていきますが、そのうちページごとにまとめます。

ubuntuにaptでmysql入れたときの話

  • ubuntu 18.04
  • mysql 5.7

パスワード聞かれずに入る。 

$ mysql -u root -p

とかでログインできない。これは、OSのユーザで入るよーという設定にされているため。パスワードも設定されていないので、これを設定する。 

$ sudo mysql  
mysql> use mysql;  
mysql> UPDATE user SET plugin='mysql_native_password' WHERE User='root';  
mysql> UPDATE user SET authentication_string=password('新規パスワード') WHERE user='root';  
mysql> FLUSH PRIVILEGES;

これで、ルートのパスワードの設定が出来る。
