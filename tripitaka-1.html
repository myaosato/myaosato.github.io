Installation of Tripitaka

Tripitakaのインストールやら何やらについてです。

どこからダウンロードできるか

GitHub上のリポジトリは、こちら https://github.com/myaosato/tripitaka

入手にはいろいろと方法があるかと思われますが、

zip形式のファイルをダウンロードする。

ページ右上の緑色のボタンを押下後、出てくるDownload ZIPをクリックして.zipファイルをダウンロード後、適当なところに展開する。

git clone する。

適当なディレクトリで、 

$ git clone https://github.com/myaosato/tripitaka.git

roswellを使う。

roswellを使っている場合は、 

$ ros install myaosato/tripitaka

以上のような方法でダウンロードしてくることが可能です。

ロードする方法について

Quicklispやら、ASDFが便利でしょう。QuicklispとASDFについての日本語の情報は、以下のリンクの先を参照すると良いかなぁと。

  • ASDF Common LISP users jp
  • Quicklisp Common LISP users jp

上記でダウンロードしてくるときにquicklisp(asdf)が読み込める場所にダウンロードしましょう。それぞれが、使用できる状態で以下のようにします。

Quicklispの場合

CL-USER> (ql:quickload :tripitaka)

ASDFの場合

CL-USER> (require :asdf)  
CL-USER> (asdf:load-system :tripitaka)

とすれば、ロードされます。
